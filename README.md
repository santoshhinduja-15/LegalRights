# LegalRights
LegalRights is a multilingual legal awareness mobile application that helps people understand important laws and rights in simple, easy-to-understand language. The app explains legal concepts clearly in **English, Hinglish, and Marathi**, making it accessible for everyone.

## Features
- Multilingual support (English, Hinglish, Marathi)
- Interactive rule-based chatbot for legal queries
- Categorized sections like Consumer Rights, Property Laws, Women’s Safety, and Cyber Laws
- Real-life scenarios for better understanding
- Modern, responsive, and animated user interface

## How It Works
1. Select a law category to explore simplified legal content.  
2. Ask any legal question in the chatbot — it provides relevant guidance.  
3. Learn applicable laws and IPC sections in your chosen language.  

## Tech Stack
- Framework: React Native (Expo)
- Icons: Ionicons, MaterialCommunityIcons, FontAwesome5
- UI: StyleSheet + LinearGradient
- Animations: React Native Animatable

## Installation
To run the project locally:
- git clone https://github.com/santoshhinduja-15/LegalRights.git
- cd LegalRights
- npm install
- npm start

## Purpose
LegalRights aims to promote legal literacy by breaking complex laws into simple explanations that everyone can understand — empowering users to know their rights and take informed actions.
